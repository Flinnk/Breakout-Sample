# Breakout-Sample

## Description

Breakout (Arkanoid) sample game for the [Simple Game Engine](https://github.com/Flinnk/Simple-Game-Engine).

## Controls

A - Move paddle left

D - Move paddle right

Spacebar - Launch ball at start
